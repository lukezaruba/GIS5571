Section 1: Overview

Originator:Metropolitan Council

Title: Functional Class Roads - Existing

Abstract: The functional class roads layer contains major roads and highways in the Twin Cities seven county metropolitan area.

In 2021, MnDOT did a full review of the metro area functional class system at the direction of the Federal Highway Administration. The result of this review was a large scale change of the metro functional class, including addition of many collectors. This file is markedly different than the version dated 2020. Questions about this process should be directed to metadata contact.

As of Sept 2019, the line work for this dataset is from the Metro Regional Centerline Collaborative (MRCC). The previous versions were based on the NCompass data. Data has been reviewed for errors and dropped links, but no guarantee is made in accuracy for minor linework errors. Please report any issues found in linework to Rachel Wiken (contact on Metadata).

The specific classifications assigned to the roadways resulted from efforts of the Land Transportation Committee, Transportation Technical Advisory Committee and the Transportation Advisory Board as specified in the transportation planning process, using federal rules and guidelines. The final product was approved with Metropolitan Council concurrence.

The 'A' minor arterial network was adopted by the Transportation Advisory Board in 1993.

Attributes include functional classification code and road name.

Please note that changes to the status of the planned functional class roads MAY impact the classification of this layer. Please confer with Metropolitan Council staff if you have questions.

Purpose: To assist the Twin Cities Metropolitan Planning Organization with transportation planning and project programming in the Twin Cities region.
